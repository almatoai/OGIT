ogit/MARS/Software

Vertices of this type are the 'S'-nodes in the MARS model. 'S' stands for 'Software' and those nodes form the second lowest layer in the A-R-S-M layer model. It's up to the user whether it will model a specific instance of a software (linked to just one 'Machine' node) or several identical instances (then the 'Software' node will be linked to several 'Machine' nodes
