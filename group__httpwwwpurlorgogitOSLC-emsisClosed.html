ogit/OSLC-ems/isClosed

This boolean property indicates if the project is closed. No further activities or measurements are done on closed projects. The measurements of closed projects can be used to calibrate the estimates for new projects. When a project is completed and all measurements on it have been performed, it is marked as closed by setting this property to true.
