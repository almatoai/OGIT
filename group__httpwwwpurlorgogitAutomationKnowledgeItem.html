ogit/Automation/KnowledgeItem

Formalized Knowledge stored in XML format. Serves as the building block for automation solutions.

A KnowledgeItem is a knowledge module, which is applied for execution of an issue. A KnowledgeItem checks therefor the information and decides if it will be executed. In that case, the KnowledgeItem executes operations, which serves the exploitation of information.
